# Product Requirements

## User Stories

### 1. User Registration
- As a new user, I want to register with my email and password so that I can create an account and access personalized features.

### 2. User Login
- As a registered user, I want to log in with my email and password so that I can access my dashboard and manage my watchlist.

### 3. Watchlist Management
- As a logged-in user, I want to add crypto symbols to my favorites (watchlist) so I can track them easily.
- As a logged-in user, I want to remove crypto symbols from my watchlist so I can manage my tracked assets.
- As a logged-in user, I want to view my current watchlist on my dashboard.
